# Viontin Gems

> **Experimental Project** — This is an experimental project under active development.

Plugin system for the Viontin framework.

## Gems

| Crate | Description |
|-------|-------------|
| `viontin-gems` | Core gem types, registry, validator |
| `viontin-gem-tailwind` | TailwindCSS build-time CSS generation |
| `viontin-gem-inertia` | InertiaJS-compatible server adapter |
